Associate Lookup
This tool allows you to look up an associate by name to find their employee ID and birth year. Helpful for helping an associate sign in to the site or for signing an associate into a session manually.
Open In New Tab
Manual Session Sign In
This tool allows you to manually sign an associate into a session by providing their employee ID and the session code of the class you want to sign them into. Note that it won't show any difference in the database when using this tool and the sign in time will be recorded as the current time when you submit the form.
